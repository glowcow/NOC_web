Project is created with:
* PostgreSQL 13
* PHP-FPM 7.4
* CSS & Sass
* JavaScript
    
## Setup
To run this project, run ```sudo docker run -t -p 80:80 --name noc_web --restart unless-stopped glowcow/noc_web:latest``` .
